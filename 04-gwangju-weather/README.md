# 04 · 광주 날씨 앱

> 외부 라이브러리 없이 순수 HTML/CSS/JS로 만든 날씨 위젯

## 개요

Vanilla JavaScript로 날씨 UI를 구성한 정적 웹 페이지입니다.  
번들러나 프레임워크 없이 브라우저에서 바로 실행됩니다.

## 파일 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 날씨 앱 단일 파일 |

## 실행 방법

```bash
# 브라우저에서 파일 직접 열기
start 04-gwangju-weather/index.html
```

## 배운 점

- CSS Grid / Flexbox 레이아웃 실습
- 외부 의존성 없이 구현하는 Vanilla JS 접근법
